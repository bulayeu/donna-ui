# donna-ui

[Storybook](https://bulayeu.github.io/donna-ui/)

Init

```npm install``` 

Or

```yarn```

Run watchers

```npm run dev```

Create docs and library build

```npm run build```

Build lib only

```npm run dist-lib```

Build docs only

```npm run dist-docs```

Watch lib only

```npm run watch-lib```

Watch docs only

```npm run watch-docs```
